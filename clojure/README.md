#Clojure Tic-Tac-Toe

**Quick Start**  
Launch a Human vs Computer game with lein run 

**CLI Options**
Option    | Value
------    | -----
Help      | lein run -- -h  
Game type | -t #{0 1 2 3}  
P1 token  | -f #{all non-numeric single characters}  
P2 token  | -s #{all non-numeric single characters}  

**Unit Tests**  
Start the SpeclJ test suite with lien spec
